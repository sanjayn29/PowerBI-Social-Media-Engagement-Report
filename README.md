# Power BI Social Media Engagement Report

This project analyzes social media engagement data using Power BI. The report provides insights into various engagement metrics from the provided dataset.

## Dataset

The analysis is based on the `Social Media Engagement.csv` file, which contains data about social media posts and their engagement statistics.

## Report

The Power BI report file (`Social Media Engagement Report.pbix`) visualizes the key findings from the dataset. You can open this file in Power BI Desktop to explore the interactive dashboards.
